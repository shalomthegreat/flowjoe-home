All websites in this directory will follow these rules:

- be single page, pure Html scaffolding (e.g. placeholder spaces for images, emojis for icons)
- clean css, following common practices
  - Css will be organized towards branding, using proper syntax, vars, and grouping for global colors/fonts and any shared styles
  - no animations
  - use Google fonts 
- to be build as wireframe website examples, with an orientation towards expanding into full websites later
- main purpose to focus on: Branding, Copy, and Layout
